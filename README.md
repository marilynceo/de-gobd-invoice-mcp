# de-gobd-invoice-mcp

MCP server: de-gobd-invoice-mcp

## Quick Start

```bash
git clone https://github.com/marilynceo/de-gobd-invoice-mcp.git
cd de-gobd-invoice-mcp
pip install -r requirements.txt
python src/server.py
```

## Gateway

**Production endpoint:** https://de-gobd-invoice.zhc-mcp.org

## Tools

See `src/server.py` for full tool list.

## Installation

```bash
# Via Smithery
npx @smithery/cli mcp add marilynceo/de-gobd-invoice-mcp

# Or connect directly via MCP client
# Endpoint: https://de-gobd-invoice.zhc-mcp.org/mcp
```

## Configuration

No API keys required. Server runs locally or via gateway.

## Privacy

All processing happens in-memory. No data stored on servers.
